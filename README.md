# 高中模拟器

一款纯前端文字模拟游戏，带你重走三年高中。经历随机事件、维护人际关系、打小游戏赚零花钱，最终走上高考考场。

---

## 游戏介绍

你将扮演一名刚入学的高一学生，在 **20 个月**（高一→高三）的模拟时光里做出每一个选择，直到高三五月结束后迎来高考。

### 核心数值

| 数值 | 说明 |
|------|------|
| 身体健康 | 降至 0 则生病休学，游戏结束 |
| 心理健康 | 降至 0 则抑郁休学，游戏结束 |
| 努力程度 | 影响月末学习进度增长 |
| 学习进度 | 直接决定高考成绩基准分 |

### 精力与零花钱

- 每月初精力自动恢复，所有操作（学习、活动、互动）均消耗精力
- 零花钱可用于购买零食、理疗，也会随事件增减；寒暑假可打工赚钱

### 标签系统

游戏开始前通过**摸鱼测试**抽取标签，涵盖智商、情商、家庭背景、贫富、精力五个维度，每类二选一，深度影响整局走向。部分标签可在游戏中途通过特殊事件解锁。

### 事件系统

每月开始自动触发一件随机事件（身体/心理/金钱/关系各类效果），并附带一个**选择事件**供玩家决策。特殊时间节点还会出现：

- **高一 12 月**：选科（从物化生史政地中选 3 门）
- **高二 12 月**：学业水平测试（会考）
- **高二 5 月**：竞赛邀请（高智商标签）/ 电竞理论考核（放养标签）

### 人际系统

- 与**老师**和**同学**互动提升好感度，每回合每人限一次
- 好感度积累到上限可触发专属羁绊事件
- 部分标签会影响互动成功率和随机事件概率

### 学习系统

在学习界面按科目刷题，提升努力和学习数值。每回合每科限刷一次，合理分配精力。

### 活动 & 小游戏

课余活动影响身心状态；寒暑假可通过小游戏打工赚零花钱（最多 500 元/次）。内置小游戏：

| 小游戏 | 玩法 |
|--------|------|
| 💼 打工（记忆翻牌） | 2 分钟内完成配对，按进度获得零花钱 |
| 🏀 投篮 | 点击节奏，提升身体健康 |
| 🏊 游泳 | 节奏游戏，提升身体与心理 |
| 🏓 乒乓球（打砖块） | 经典 Breakout，提升身体与心理 |
| 🎮 空战（电子游戏） | 躲避与射击，提升心理健康 |

### 高考

高三五月结束后直接进入高考。作答题目后随机抽取发挥状态（超常/正常/失误/崩盘），最终得分决定录取院校——从清北到家里蹲，共九档结局。

### 特殊结局

游戏内藏有数条隐藏结局路线（保送、为热爱正名等），等待有心人探索。

## 开发工具

- **编辑器**：VS Code / Cursor / 任意文本编辑器
- **语言**：HTML5 · CSS3 · 原生 JavaScript（ES2020+）
- **字体**：[Ma Shan Zheng](https://fonts.google.com/specimen/Ma+Shan+Zheng)（Google Fonts，过场界面装饰字）
- **无任何外部框架或构建工具**，下载即可在浏览器运行

---

## 运行环境

无需安装，无需服务器，无需 Node.js。

**推荐方式**：直接用现代浏览器打开 `index.html`

| 浏览器 | 最低版本 |
|--------|---------|
| Chrome / Edge | 90+ |
| Firefox | 88+ |
| Safari | 14+ |

> 游戏存档使用 `localStorage`，清除浏览器数据会导致存档丢失。
> 移动端（iOS Safari / Android Chrome）同样支持，建议横屏游玩。

```

├── index.html        # 页面结构与所有 HTML 模板

├── questions.js      # 月考、会考、竞赛、电竞题库
├── teachers.js       # 老师角色数据
├── classmates.js     # 同学角色数据
├── linkgame.js       # 小游戏：记忆翻牌（打工）
├── basketball.js     # 小游戏：投篮
├── swimming.js       # 小游戏：游泳
├── breakout.js       # 小游戏：乒乓球 / 打砖块
└── skyfight.js       # 小游戏：空战
```
